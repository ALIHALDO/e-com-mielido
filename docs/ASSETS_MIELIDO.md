## 2. Usage recommandé par section

### Header

- Utiliser `logo horizontal.png`.
- Prévoir une version compacte avec `icone.png` sur mobile si nécessaire.

### Hero section

- Utiliser `logo principale.png` ou `ChatGPT Image 23 mai 2026, 17_33_36.png`.
- Ajouter un fond ivoire chaud avec motif `motif.png` en opacité faible.

### Section gamme

- Utiliser `ChatGPT Image 23 mai 2026, 17_33_36.png` comme référence de présentation de la gamme.
- Si des photos produit isolées sont disponibles plus tard, les utiliser dans les cartes produits.

### Section marque

- Utiliser `co-branding.png` pour montrer le lien entre Mielido et Al’Fat Entreprise.
- Utiliser `signature.png` pour rappeler la promesse de marque.

### Section revendeur

- Utiliser `sticker.png` ou `embleme.png` pour donner une sensation premium et professionnelle.

### Footer

- Utiliser `logo horizontal.png` ou `co-branding.png`.

## 3. Règles de performance

Avant intégration web :

- compresser les images ;
- générer des versions WebP ;
- prévoir des tailles desktop et mobile ;
- renseigner les attributs alt ;
- éviter les images trop lourdes en hero mobile.

## 5. Alt text recommandé

- `Logo Mielido horizontal, miel premium par Al’Fat Entreprise`
- `Logo principal Mielido avec abeille et goutte de miel`
- `Gamme finale Mielido avec formats 0,25 L, 0,5 L, 1 L et 1,5 L`
- `Motif alvéoles dorées Mielido`
- `Emblème Mielido avec goutte de miel et abeille`
- `Co-branding Mielido par Al’Fat Entreprise`

## 6. Règles de cohérence

- Ne pas mélanger plusieurs styles de logo dans une même section.
- Ne pas utiliser le motif alvéoles en pleine opacité derrière du texte important.
- Ne pas déformer les logos.
- Garder une hiérarchie claire : Mielido d’abord, Al’Fat Entreprise en soutien.
