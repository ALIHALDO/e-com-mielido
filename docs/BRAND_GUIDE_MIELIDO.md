# Brand guide Mielido — Boutique en ligne

## 1. Essence de marque

**Mielido** est une marque de miel portée par **Al’Fat Entreprise**.

Signature : **L’Or de la nature, pur & authentique**

La marque doit transmettre une sensation de douceur, de naturalité, de confiance et de valeur premium.

## 2. Positionnement visuel

Mielido se positionne comme une marque :

- premium mais accessible ;
- naturelle mais moderne ;
- locale mais professionnelle ;
- familiale mais élégante ;
- pratique grâce aux formats squeeze.

## 4. Combinaisons recommandées

### Combinaison premium

- Fond : `#27123D`
- Texte : `#FFF6E6`
- CTA : `#D4A017`
- Détails : `#F2C46D`

### Combinaison claire

- Fond : `#FFF6E6`
- Texte : `#27123D`
- Accent : `#D4A017`
- Cartes : `#EADDC7` avec faible opacité

### Combinaison produit

- Fond : ivoire chaud
- Carte : blanc/ivoire
- Bordure : or miel
- Prix : prune royal
- Badge format : prune royal avec texte or miel

## 5. Typographies

Usage :

- H1 ;
- grands titres ;
- citations ;
- signature de marque ;
- titres de sections premium.

Style : élégant, luxe, raffiné.

### Montserrat

Usage :

- paragraphes ;
- navigation ;
- boutons ;
- fiches produits ;
- prix ;
- formulaire ;
- FAQ.

Style : moderne, lisible, professionnel.

## 7. Règles d’utilisation du logo

- Utiliser le logo horizontal dans le header.
- Utiliser le logo principal dans le hero ou la section marque.
- Utiliser le co-branding dans le footer ou la page À propos.
- Ne pas étirer le logo.
- Ne pas changer les couleurs du logo.
- Ne pas placer le logo sur un fond qui réduit sa lisibilité.
- Garder une marge de sécurité autour du logo.

## 8. Motifs et icônes

Le motif alvéoles doit rester subtil.

Utilisations recommandées :

- arrière-plan de section ;
- séparateur ;
- cadre de carte produit ;
- bloc revendeur ;
- bannière promotionnelle.

L’icône abeille/goutte doit être utilisée avec modération pour éviter la surcharge visuelle.

## 9. Style des boutons

### Bouton principal

- Fond : `#D4A017`
- Texte : `#27123D`
- Hover : `#F2C46D`
- Coins arrondis : 12 à 18 px
- Texte : Montserrat semi-bold

Exemples :

- Commander maintenant
- Commander sur WhatsApp
- Ajouter au panier

### Bouton secondaire

- Fond transparent ou ivoire
- Bordure : `#D4A017`
- Texte : `#27123D`

Exemples :

- Découvrir la gamme
- Demander le tarif revendeur

## 10. Style des cartes produits

Chaque carte produit doit contenir :

- image claire ;
- badge format ;
- nom du produit ;
- prix visible ;
- prix revendeur ;
- CTA ;
- coins arrondis ;
- bordure dorée fine ;
- ombre douce.

## 11. Ton rédactionnel

Ton Mielido :

- doux ;
- premium ;
- rassurant ;
- naturel ;
- familial ;
- direct ;
- commercial sans agressivité.

Exemples :

- “Une touche dorée pour vos petits-déjeuners.”
- “Des formats pratiques pour chaque besoin.”
- “Commandez Mielido en quelques secondes sur WhatsApp.”
- “Le miel pratique pour la maison, le thé et la cuisine.”

## 12. Interdits de marque

Ne pas utiliser :

- couleurs néon ;
- polices fantaisie non prévues ;
- promesses médicales ;
- textes trop agressifs ;
- faux labels qualité ;
- visuels pixellisés ;
- mises en page trop chargées ;
- incohérences entre Mielido et Al’Fat Entreprise.
